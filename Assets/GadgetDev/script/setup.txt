新增Gadget
    1.新增欲新增會影響到player數值的function東西至IEffecter
    2.從player/model 裡面新增一個取得或更改player數值的function
    3.新增一個script 繼承BaseGadget or basepotion ex:Heal.cs
    4.從剛剛新增的script中protected override void CallWhenUse()
    5.再CallWhenUse()裡面寫欲新增之(可存取base.ownerEff)
    6.new prefab ->add Heal and setup
    7.新增一個spawner到場景上的某個位置，將要產生的prefab放入control->spawn gadget
玩家設定

    1.新增pickGadget.cs到player裡
    2.在按鍵的地方，using Gadget.Pick;
    3.設定使用東西的按鍵，呼叫 <PickGadget>.UseGadget()
    4.新增slot ui
    5.設定pickGadget的slot ui
